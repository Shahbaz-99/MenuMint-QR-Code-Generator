# MenuMint 🍽️📱

**MenuMint** is a minimal and modern QR code generator built with Django.  
It helps restaurants create and share digital menus instantly using QR codes.

---

## 🚀 Features

- ⚡ Instant QR code generation
- 📱 Contactless dining support
- 🎯 Clean and modern UI
- 📥 Download high-quality QR codes (PNG)
- 🗂️ Stores QR images in media folder
- 🔗 Easy sharing via menu links

---

## 🛠️ Tech Stack

- Python
- Django
- qrcode, pillow library
- HTML / CSS
